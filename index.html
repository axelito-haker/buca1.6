Software Sistema formal de componentes lógicos necesarios para que funcione un sistema informático. The formal system of a computer, consisting of the necessary logical components for operation.
Hardware Parte física de un ordenador o sistema informático. Physical part of a computer or computer system.
TIC's Permiten la compilación, almacenamiento y transmisión de información como voz, datos, textos e imágenes. They allow the compilation, storage, and transmission of information such as voice, data, texts, and images.
Web Sistema de distribución de información basado en hipertexto o hipermedios accesibles por internet. Information distribution system based on hypermedia accessible through the internet.
Redes Informáticas Conjunto de equipos conectados para compartir información y recursos. Set of equipment connected to share information and resources.
Protocolo Sistema de reglas que permiten que más entidades de un sistema de comunicación se comuniquen entre ellas para transmitir información por medio de cualquier tipo de variación de una magnitud física. A system of rules that allows for the communication of more entities of a communication system, transmitting information through any type of physical variation of a magnitude.

Código Un conjunto de palabras o símbolos que contienen instrucciones para la computadora. A set of words or symbols that contain instructions for the computer.

Procesadores Dispositivos electrónicos que realizan cálculos y operaciones en los ordenadores. Electronic devices that perform calculations and operations on computers.

Mouse Dispositivo que permite al usuario interactuar con el ordenador. Device that allows the user to interact with the computer.

Internet Red global que conecta a los ordenadores y permite la comunicación. Global network that connects computers and allows communication.

Tecnología Conocimientos y habilidades para desarrollar y mantener sistemas informáticos. Knowledge and skills to develop and maintain computer systems.
